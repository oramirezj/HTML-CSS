My Skills.

Web Developer

A full-stack web developer is responsible for designing, developing, and maintaining all aspects of a web application. This includes the front-end (client-side) using technologies such as HTML, CSS, and JavaScript, as well as the back-end (server-side) using technologies such as PHP, Ruby on Rails, or Python. They also have experience with databases and the infrastructure needed to run a web application. In short, a full-stack web developer is able to handle all aspects of creating and maintaining a web application.

Data Analyst

A data analyst is responsible for collecting, processing, and analyzing large sets of data to extract valuable insights and information. They use various tools and techniques such as statistical analysis, data visualization, and machine learning to uncover patterns and trends in the data. They also work closely with other teams such as business and IT, to interpret data and provide recommendations based on their findings. In short, a data analyst is responsible for turning raw data into actionable insights to help inform business decisions.
